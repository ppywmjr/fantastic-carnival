# fantastic-carnival

This is a Jekyll-based GitHub Pages example site using the minima theme.

## Getting Started

### Prerequisites

- Ruby (recommended: use rbenv)
- Bundler (`gem install bundler`)

### Setup

1. Clone the repository:
	```sh
	git clone https://github.com/ppywmjr/fantastic-carnival.git
	cd fantastic-carnival
	```
2. Install dependencies:
	```sh
	bundle install
	```

### Running Locally

Start the Jekyll development server:

```sh
bundle exec jekyll serve
```

Visit [http://localhost:4000](http://localhost:4000) in your browser to view the site.

## Writing Blog Posts

Add new markdown files to the `_posts/` directory using the format `YYYY-MM-DD-title.md`.

## Deployment

Push changes to the `main` branch. GitHub Pages will automatically build and deploy the site.

---
For more details, see the [Jekyll documentation](https://jekyllrb.com/docs/) and [minima theme](https://github.com/jekyll/minima).
